# 🚀 Proyecto Alros Inmobiliaria

¡Hola! Este es el proyecto en el que estoy trabajando para la inmobiliaria Alros. Es una aplicación web moderna construida con **Next.js**, **Tailwind CSS** y **Supabase**.

## 🛠️ Cómo ponerlo en marcha

Para poder ver la web en tu ordenador, sigue estos sencillos pasos:

### 1. Requisitos
Necesitas tener instalado **Node.js** (versión 18 o superior). Si no lo tienes, puedes descargarlo en [nodejs.org](https://nodejs.org/).

### 2. Instalación
Abre una terminal (o consola) dentro de esta carpeta y escribe:
```bash
npm install
```
*Esto descargará todas las librerías necesarias (tardará un par de minutos).*

### 3. Ejecución
Una vez termine la instalación, escribe:
```bash
npm run dev
```

### 4. ¡Listo!
Abre tu navegador y entra en la dirección:
`http://localhost:3000`

---

## 🔑 Qué vas a ver:
- **Página Principal**: Rediseñada con estética premium.
- **Buscador de Propiedades**: Con filtros avanzados de precio, habitaciones y tipo.
- **Panel de Administración**: Entra en `http://localhost:3000/admin` para ver la gestión interna, estadísticas reales y el creador de inmuebles con IA.
- **Mapa Interactivo**: Ubicación de las propiedades sobre Google Maps.
