# Flujo de Desarrollo Autónomo

Este flujo permite al asistente realizar las siguientes acciones sin requerir confirmación explícita para cada comando:

1.  **Investigación del código**: Ejecutar comandos de búsqueda (grep, find, ls) para entender la estructura.
2.  **Pruebas y Verificación**: Ejecutar lints, tests y comandos de arranque (npm run dev, etc.) para validar cambios.
3.  **Generación de Archivos**: Crear scripts de utilidad o archivos temporales necesarios para la tarea.
4.  **Despliegue y Sincronización**: Comandos de git o herramientas de despliegue si se solicita.

El usuario ha autorizado este modo para maximizar la eficiencia y evitar bloqueos por esperas de aprobación.
